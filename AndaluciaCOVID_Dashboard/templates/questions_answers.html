FAQ

¿De qué fuentes se obtienen los datos?

Los datos se obtienen de diferentes fuentes, pero siempre son datos oficiales respaldados por el Ministerio de Sanidad y la Junta de Andalucía (IECA).

¿Puedo utilizar los datos en mi propio proyecto?

Sí, además de tener disponibles las fuentes oficiales mencionadas anteriormente puedes usar la API disponible aquí para obtener los datos en formato JSON.

No me funciona el modo oscuro aunque lo tenga configurado en mi dispositivo

Esta funcionalidad es compatible con todos los navegadores modernos, básicamente Firefox y Chrome así como los basados en Chromium como Opera, Brave o Microsoft Edge. Incluso en sus versiones móviles podrás ver la página web en modo oscuro.
